Mengapa bahasa baik ?

  1. Kode yang ditulis dalam bahasa Indonesia.
  2. Tidak perlu dikompile.
  3. Ekstensi kode BAIK adalah .ina yang berarti Indonesia.
  4. Variable pada BAIK memiliki tipe otomatis sehingga tidak perlu mendeklarasikan variable terlebih dahulu.
  5. Array/Larik (dalam BAIK disebut Untaian) hanya ada dalam satu dimensi.
  6. Fungsi/Sub Prosedur bisa didefinisikan sebelum maupun sesudah kode utama.

Mengapa bahasa baik ?

  1. Dapat mengakses basisdata.
  2. Dapat membuat file gambar dan grafik
  3. Baik dapat digunakan pada Web System melalui CGI.
  4. Dapat digunakan untuk membuat program jaringan dengan TCP/IP
  5. Dapat digunakan untuk membuat program desktop
  6. Dapat digunakan untuk pemrograman ROBOT (micro Controller) raspberry
